# v1.0.0 — Initial MVP release

Первый публичный релиз ML-системы предиктивной диагностики СВЧ-модулей.

## Что входит в релиз

- генерация синтетических временных рядов датчиков;
- оконная обработка сигналов;
- извлечение температурных, вибрационных, частотных и режимных признаков;
- обучение baseline-модели;
- обучение ансамблевых ML-моделей;
- выбор лучшей модели по F1-macro;
- оценка качества классификации;
- матрица ошибок;
- Streamlit-интерфейс;
- роли оператора, инженера-техника и ML-инженера;
- оценка риска;
- формирование предупреждений;
- план проверки оборудования.

## ML-часть

- задача: многоклассовая классификация;
- классов: 7;
- лучшая модель: Extra Trees;
- accuracy: около 0.87;
- F1-macro: около 0.87.

## Ограничения

- данные синтетические;
- система требует проверки на реальных промышленных данных;
- для неизвестных дефектов нужен отдельный модуль anomaly detection.
